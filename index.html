Presto is a high performance, distributed SQL query engine for big data.

Download GitHub

High performance

Presto is a highly parallel, distributed query engine, built from the ground up for efficient, low latency analytics.

Versatile

Presto supports diverse use cases: ad-hoc analytics at interactive speeds, massive multi-hour batch queries, and high volume applications that perform sub-second queries.

In-place analysis

Presto can natively query data in Hadoop, S3, Cassandra, MongoDB, MySQL, PostgreSQL and many other data stores, without the need for complex and error-prone processes for copying the data to a proprietary storage system.

Query federation

Presto can access data from multiple systems within a single query. For example, it can join historic log data stored in S3 with real-time customer data stored in MySQL.

Works with existing BI tools

Presto is an ANSI SQL compliant query engine and works with BI tools such as R, Tableau, MicroStrategy, Power BI, and Superset.

Trusted

Presto is used for for critical business operations, including financial results for public markets, by some of the largest organizations in the world.

Runs everywhere

Presto is optimized for both on-premise and cloud environments such as Amazon, Azure, and Google Cloud.
